json-server

Manual de JSON Server

json-server es una herramienta que te permite crear una API RESTful completa con un archivo JSON como base de datos. Es ideal para prototipos rápidos, pruebas y desarrollos sin necesidad de configurar un backend completo.

Instalación

Para instalar json-server, necesitas tener Node.js y npm instalados. Luego, puedes instalar json-server globalmente con el siguiente comando:

npm install -g  json-server@0.17.1

Documentación

https://www.npmjs.com/package/json-server/v/0.17.4

Creación de la Base de Datos

Crea un archivo llamado db.json que contendrá tu base de datos en formato JSON. Aquí tienes un ejemplo básico:

{
  "posts": [
    { "id": 1, "title": "Hello World", "author": "John Doe" },
    { "id": 2, "title": "JSON Server", "author": "Jane Doe" }
  ],
  "comments": [
    { "id": 1, "body": "Nice post!", "postId": 1 },
    { "id": 2, "body": "Thanks!", "postId": 1 }
  ]
}

Iniciar el Servidor

Para iniciar el servidor, ejecuta el siguiente comando en el directorio donde se encuentra tu archivo db.json:

json-server --watch db.json --port 4000

Esto iniciará el servidor en http://localhost:4000 por defecto.

Operaciones CRUD

Crear (POST)

Para crear un nuevo recurso, envía una solicitud POST a la URL correspondiente. Por ejemplo, para crear un nuevo post:

POST /posts
{
  "title": "Nuevo Post",
  "author": "Autor"
}

Leer (GET)

Para obtener todos los posts, envía una solicitud GET a /posts:

GET /posts

Para obtener un post específico por ID:

GET /posts/1

Actualizar (PUT/PATCH)

Para actualizar un recurso existente, envía una solicitud PUT o PATCH a la URL correspondiente. Por ejemplo, para actualizar un post:

PUT /posts/1
{
  "title": "Post Actualizado",
  "author": "Autor Actualizado"
}

Eliminar (DELETE)

Para eliminar un recurso, envía una solicitud DELETE a la URL correspondiente. Por ejemplo, para eliminar un post:

DELETE /posts/1

Rutas Adicionales

json-server también soporta rutas adicionales para consultas más avanzadas. Por ejemplo:

GET /posts?author=John Doe
GET /posts?_sort=title&_order=asc
GET /posts?_page=1&_limit=10

Conclusión

json-server es una herramienta poderosa y fácil de usar para crear APIs RESTful rápidas y prototipos. Con este manual básico, deberías poder empezar a utilizarlo en tus proyectos.
